# Smart Environmental Data Logger
**ESP32-C3 + ESP RainMaker + FreeRTOS**

## Features
- ✅ FreeRTOS multi-tasking (5 concurrent tasks)
- ✅ DHT11 Temperature & Humidity monitoring
- ✅ Push notifications for critical alerts
- ✅ Voice control (Alexa/Google Assistant)
- ✅ OTA firmware updates
- ✅ Espressif Insights dashboard

## Hardware
- ESP32-C3-MINI-1 Development Board
- DHT11 Sensor
- 5V DC Cooling Fan (via Relay)

## Software
- Platform: ESP-IDF / Arduino Framework
- Cloud: ESP RainMaker
